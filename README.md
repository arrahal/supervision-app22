## 🚀 تشغيل التطبيق (GitHub Pages)

### الخطوة 1 — رفع الملفات على GitHub
```bash
git clone https://github.com/YOUR_USERNAME/superviseur-app.git
cd superviseur-app

# انسخ محتوى هذا ZIP إلى المجلد
git add .
git commit -m "🎉 Initial commit - نظام المشرف التربوي"
git push origin main
```

### الخطوة 2 — تفعيل GitHub Pages
1. افتح **Settings** في مستودعك
2. انتقل إلى **Pages**
3. اختر **Source: Deploy from a branch**
4. اختر **Branch: main** → **/ (root)**
5. انقر **Save**

### الخطوة 3 — تثبيت على الهاتف
بعد دقيقتين، افتح الرابط على هاتفك:
```
https://YOUR_USERNAME.github.io/superviseur-app/
```
ستظهر رسالة **"إضافة إلى الشاشة الرئيسية"** — انقر عليها!

---

## 📱 تثبيت التطبيق على الهاتف

### Android (Chrome)
1. افتح الرابط في **Chrome**
2. انقر على زر **"📲 تثبيت التطبيق"** في الشريط العلوي
3. أو: القائمة (⋮) → **"إضافة إلى الشاشة الرئيسية"**

### iPhone (Safari)
1. افتح الرابط في **Safari**
2. انقر على زر **المشاركة** (□↑)
3. اختر **"إضافة إلى الشاشة الرئيسية"**
4. انقر **إضافة**
